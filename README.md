# Tessl Homebrew Tap

Official Homebrew tap for [Tessl](https://tessl.io).

## Installation

```bash
brew tap tesslio/tessl
brew install tessl
```

## Usage

After installation, you can use the Tessl CLI:

```bash
tessl --help
```

## Updating

To update to the latest version:

```bash
brew update
brew upgrade tessl
```

## Uninstallation

```bash
brew uninstall tessl
brew untap tesslio/tessl
```
